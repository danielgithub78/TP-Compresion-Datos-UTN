# TP-Compresion-Datos-UTN
hecho en Python

Pasos en Debian 13
Usamos Visual Studio Code (VSC) como Editor
Entramos a VSC abrimos una terminal desde ahi.
sudo apt install python3-tk python3-venv (instalamos esas librerias para crear entornos virtuales)
python3 -m venv mi_entorno (creamos un entorno virtual cerrado)
source mi_entorno/bin/activate (activamos el entorno virtual)
pip install matplotlib (instalamos esta libreria dentro del entorno virtual)
python app.py (correr la aplicacion).
